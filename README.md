## 1. Introduction <a name="introduction"></a>
Performed in Brainster - Data Science Bootcamp.

We got an Assignment to Classify five type of images:
* Indoor Selfie
* Outdoor Selfie
* Indoor Pose
* Outdoor Pose
* Photos Without Human
